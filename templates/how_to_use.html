How to Use

Follow these steps to use the Email Sender application:

  1. Set up the SMTP server by following the link provided.
  2. Enter the sender's email address and password generated from the SMTP server.
  3. Enter the subject and body of the email.
  4. Upload an Excel file containing the email addresses of the recipients.
  5. Upload the attachments to be sent with the email.
  6. Click on the "Send Emails" button to start sending the emails.
